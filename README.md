# Mein Lebensrad – Kostenloser Selbstcheck

**Tanja Sattelberg · Systemische Beraterin & Coach**

Ein kostenloser interaktiver Selbstcheck – für Frauen über 50, die spüren wollen, wo sie wirklich stehen.

> **Hinweis:** Dieses Tool ist ein Coaching- und Selbstreflexionsangebot. Es ersetzt keine Psychotherapie und keine medizinische Behandlung.

---

## Was dieses Tool bietet

### Die 3 Tabs

| Tab | Inhalt |
|---|---|
| **Mein Lebensrad** | Alle 8 Lebensbereiche auf einer Skala von 0–10 bewerten · Interaktives Rad · Erste Reflexionsfragen |
| **Reflexion** | Pro Lebensbereich: ein Impuls, eine Reflexionsfrage, eine Affirmation · Antwortfeld · Kurs-Teaser |
| **Mein Ergebnis ✦** | Alle Bereiche sortiert nach Wert · Gezielte Impulse zu den schwächsten Bereichen · Opt-in für weitere Impulse |

### Was bewusst weggelassen wurde

Dieses Freebie ist ein Neugierigmacher – kein vollständiger Kurs. Es gibt echten Mehrwert, aber lässt eine spürbare Lücke:

- Keine vollständigen Übungen (nur erste Impulse und eine Reflexionsfrage)
- Nur eine Affirmation pro Bereich (statt drei)
- Keine Wachstumsmessung (Vorher-Nachher)
- Keine Kursbegleitung (folgt im Grundkurs)

Nach jeder Bereichs-Reflexion erscheint ein konkreter Hinweis auf den passenden Vertiefungskurs aus der Reihe „Mein Veränderungsweg".

|---|---|
| Inneres Erleben | Mehr bei mir |

## Der Opt-in-Moment

Im Tab „Mein Ergebnis" erscheint unter den zwei schwächsten Lebensbereichen ein konkreter Teaser-Text mit dem passenden Kursnamen. Darunter befindet sich das E-Mail-Opt-in-Formular.

**Versprechen an die Nutzerin:** 3 gezielte Impulse passend zu ihren zwei schwächsten Lebensbereichen – kostenlos, unverbindlich, direkt anwendbar.

> **Hinweis für die Einrichtung:** Das Opt-in-Formular muss noch mit deinem Mailerlite-Konto verbunden werden. Suche im Code nach `// Hier kannst du deinen Newsletter-Anbieter einbinden` und trage dort die Mailerlite-API oder Formular-Action ein.

## GitHub Pages aktivieren

1. Repository auf GitHub hochladen (eigenes Repository, getrennt von der Kursversion)
2. → **Settings** → **Pages**
3. Source: `main` Branch, Ordner `/` (root)
4. Speichern
5. Empfohlener Repository-Name: `lebensrad-check` → URL: `https://dein-name.github.io/lebensrad-check`
